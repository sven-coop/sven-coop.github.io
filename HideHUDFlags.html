    HideHUDFlags

    Flags used in CBasePlayer m_iHideHUD

    Values

    Name Value Description
    HIDEHUD_WEAPONS 1 Hides Weapon Selection Menu and Crosshair elements
    HIDEHUD_FLASHLIGHT 2 Hides the Flashlight element
    HIDEHUD_ALL 4 Hides all HUD elements
    HIDEHUD_HEALTH 8 Hides the Health element
    HIDEHUD_WEAPON_SELECT 16 Hides the Weapon Selection Menu element
    HIDEHUD_SUITPOWER 32 Hides the Battery(armor) element
    HIDEHUD_CUSTOM_MAPS 64 Hides Custom HUD elements from map scripts
    HIDEHUD_CUSTOM_PLUGINS 128 Hides Custom HUD elements from plugins
    HIDEHUD_CROSSHAIR 256 Hides only the Crosshair element
    HIDEHUD_AMMO 512 Hides only the Ammo element
